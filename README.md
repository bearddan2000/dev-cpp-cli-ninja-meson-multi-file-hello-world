# dev-cpp-cli-ninja-meson-multi-file-hello-world

## Description
This is a POC project to demonstrate 
ninja with meson as a build tool.

## Tech stack
- cpp
- ninja
- meson

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
